Game Of Life

Game Of Life is a cellular automaton devised by the British mathematician John Horton Conway in 1970. It's a zero-player game. The games initial state determines the evolution of the game, thus needing no input from a player. You play by creating an initial configuration(Although you can alter the game board once it's started) and observe its evolution.

The game rules

The game board is a two-dimensional grid of cells. Each cell can either be dead or alive

  • Any live cell with fewer than two live neighbours dies (referred to as underpopulation).
  • Any live cell with more than three live neighbours dies (referred to as overpopulation or overcrowding).
  • Any live cell with two or three live neighbours lives, unchanged, to the next generation.
  • Any dead cell with exactly three live neighbours will come to life.
